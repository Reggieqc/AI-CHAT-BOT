## Laptop

### Description

A high-performance laptop designed for work, study, and creative tasks.

### Key Features

- Fast processor suitable for multitasking
- Full HD display
- SSD storage for quick boot times
- Long-lasting battery

### Usage Instructions

- Charge fully before first use
- Keep ventilation areas unobstructed
- Install operating system and driver updates regularly

### Warranty

1-year warranty covering hardware manufacturing defects.

### Refund & Return

Returns allowed within 7 days only if:
